# AI Tutor Chatbot 🤖

This is a Python-based AI chatbot project I built using Flask and PyTorch.  
It has a clean UI and works similar to a basic ChatGPT interface where users can ask Python-related questions.

## Features

- Chat interface with user and bot messages
- Typing effect while the bot responds
- Stop button to cancel long responses
- Sidebar with chat history
- Suggestions panel for quick questions
- Dark modern UI (black + orange + purple theme)

## Tech Used

- Python (Flask backend)
- PyTorch (for model)
- HTML, CSS, JavaScript (frontend)

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## Note

API key is required for AI responses (not included in this repo).
